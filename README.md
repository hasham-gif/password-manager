# 🔐 Personal Password Manager

A simple command-line password manager built with Python.

## ✨ Features

- 💾 Save passwords for any website
- 👀 View all saved passwords
- 🔍 Search password by website name
- ✏️ Update existing passwords
- 🗑️ Delete passwords
- 🎯 Smart password suggestion based on website name
- 💪 Password strength checker (Weak / Medium / Strong)
- 🔐 Master password protection

## ▶️ How to Run

```bash
python passwordmanager.py
```

## 📋 Menu Options
- Save password
- View password
- Generate password
- Search password
- Update password
- Delete password
- Exit

## 🛠️ Built With

- Python 3
- No external libraries needed!
